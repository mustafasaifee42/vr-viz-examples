VR-Viz provide a high-level react components to generate 3D visualization in webVR. It combines A-Frame with React (for DOM manipulation) and D3 (for data visualizations) to generate visualization in VR. It provides a JSON syntax for generating visualizations (the concept is inspired from Vega-Lite)

**GitHub Repo: [https://github.com/mustafasaifee42/vr-viz](https://github.com/mustafasaifee42/vr-viz)**

**NPM Package: [vr-viz](https://www.npmjs.com/package/vr-viz)**

**Boilerplate for using VR-Viz can be found [here](https://github.com/mustafasaifee42/vr-viz-boilerplate)**

This project was initiated as master thesis at Aalto University. ([Thesis link here](https://aaltodoc.aalto.fi/handle/123456789/36113))

# Getting Started

## React

To get started using vr-viz first add or install the module to your existing React project. Package link: [https://www.npmjs.com/package/vr-viz](https://www.npmjs.com/package/vr-viz)

**Installation**

```
npm install --save vr-viz
```

or

```
yarn add vr-viz
```

## Usage

```
import VRViz from "vr-viz"
```
